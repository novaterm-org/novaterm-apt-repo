NovaTerm APT Repository

aarch64 packages for the NovaTerm terminal emulator

Quick Setup

Add the repository to NovaTerm or Termux:

# 1. Add the repository
echo "deb https://novaterm-org.github.io/novaterm-apt-repo stable main" \
  >> $PREFIX/etc/apt/sources.list.d/novaterm.list

# 2. Import the GPG key
curl -fsSL https://novaterm-org.github.io/novaterm-apt-repo/pubkey.gpg \
  | apt-key add -

# 3. Update package lists
apt update

Alternative (unsigned, development only)

echo "deb [trusted=yes] https://novaterm-org.github.io/novaterm-apt-repo stable main" \
  >> $PREFIX/etc/apt/sources.list.d/novaterm.list
apt update

Repository Structure

  • dists/stable/main/binary-aarch64/ — arm64 packages
  • dists/stable/main/binary-all/ — architecture-independent packages
  • pool/main/ — package pool
  • pubkey.gpg — GPG signing key
